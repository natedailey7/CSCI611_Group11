# How to run the code for this project

To run the code for this project, simply open the Python Notebook titled `rag_cross_domain_v2-4.ipynb` in Google Colab or any code editor with Jupyter Notebook capabilties. The data and model are imported within the notebook (note the the script automatically pip installs the required packages). It is recommended to use a virtual Python environment in order to keep installations organized.

It is also recommended to run the notebook on a GPU (this script ran in about 2 hours using a GTX 1650). If you would like to run a scaled-down version of the notebook, you may reduce the number of `n` iterations. We used `n=200` for our analysis.
